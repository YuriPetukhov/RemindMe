Application Purpose:

The primary goal of this application is to assist users in effectively memorizing information. Users input the data they need to remember, along with its meaning, and these details are sent to their Telegram account at scheduled intervals.

Reminder Intervals:

The reminder intervals double after each successful recall. If the user responds incorrectly, the intervals reset, and the repetition starts from the beginning. An incorrect response at any stage prompts a repeat of the previous interval in the next cycle. There are 10 repetition intervals in total, ranging from 20 minutes to 60 days.

Statistics Interface:

The web interface provides statistics that allow users to track the distribution of cards across the reminder intervals, as well as the count and percentage of incorrect responses for each interval.

Practice Tab:

The Practice tab offers users the chance to practice their memorization skills, with words selected randomly for review.

Main Interface Page:

The main interface page displays 60 randomly selected user cards.

Continuous Improvement:

The application is continuously refined and expanded with new features to enhance both the user experience and the effectiveness of the memorization process.
